# Customer Shopping Behavior Analysis

## Project Overview
This project analyzes customer shopping behavior using transactional data from 3,900 purchases.
The objective is to uncover insights into customer spending patterns, product preferences,
discount behavior, and subscription trends.

## Dataset Summary
- Rows: 3,900
- Columns: 18
- Key features include demographics, purchase details, discounts, reviews, and subscriptions
- Missing values handled using median imputation

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib)
- PostgreSQL (SQL)
- Power BI

## Analysis Performed
- Revenue by gender
- High-spending discount users
- Top products by rating
- Shipping type comparison
- Subscriber vs non-subscriber analysis
- Customer segmentation
- Top products per category
- Revenue by age group

## Power BI Dashboard
An interactive dashboard was created to visualize customer behavior and revenue insights.

## Business Recommendations
- Promote subscriptions to improve loyalty
- Optimize discount strategy
- Focus marketing on high-revenue customer groups
